# 0.1.2
* Added experimental support for integers of different bit widths (i32, i16, i8).
* All integers must now have a prefix of their type prepended.
* Added experimental support for floating point types (f64, f32).

# 0.1.1
* Added support for variables with the procedures `set` and `fetch`.
* Added arithmetic procedures `sub`, `mul`, `div`, and `mod`.
